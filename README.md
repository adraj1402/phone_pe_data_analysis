# phone_pe_data_analysis
Examined the Phone Pe dataset using Python.
Loaded the dataset with help of PANDAS library.
Applied the statistical summary (like mean, mode, median) for numerical columns by NUMPY library.
Calculated the most frequent transaction type for each state and quarter.
Evaluated the total number of transaction and total transaction amount for each state over the years.
Identified the district with highest population for each state and created a bar chart depicting the district with highest population for each state.
